# NOIR

**A House of Invisible Luxury.**

Perfume cannot be seen. It cannot be touched. It cannot be photographed.
So this site never tries to show perfume — it visualizes memory.

A cinematic digital exhibition in nine chapters, built around *The Ether*:
a single persistent particle field that becomes every form the film needs —
dust, a bottle, raw ingredients, a molecule — before collapsing back to the
one mote where it began.

## Run

```bash
npm install
npm run dev     # localhost:3006
```
